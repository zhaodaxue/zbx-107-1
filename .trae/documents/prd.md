## 1. 产品概述

城管热线「口袋公园噪声投诉」时段复盘页，用于分析和复盘口袋公园的噪声投诉数据，帮助城管部门和公园管理人员识别噪声问题热点时段和区域。
- 核心目标：通过多维度可视化分析投诉数据，识别投诉升温趋势，辅助管理决策
- 目标用户：城管部门工作人员、公园管理人员、城市环境治理分析师

## 2. 核心功能

### 2.1 功能模块

1. **数据概览区**：筛选器（周末/施工）、日期区间选择、导出按钮
2. **周汇总柱状图**：各公园按周汇总投诉次数，支持多园对比
3. **环比分析表**：本周 vs 上周同园数据对比，增幅超40%标记「升温」
4. **时段矩阵**：公园×时段交叉矩阵，展示各时段投诉次数，升温格标橙
5. **数据导出**：框选日期区间导出明细CSV

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 复盘主页 | 顶部筛选区 | 周末筛选、施工筛选、日期区间选择器、CSV导出按钮 |
| 复盘主页 | 周汇总柱状图 | 按周聚合各公园投诉次数，支持多选公园对比，图例交互 |
| 复盘主页 | 环比分析表 | 展示各公园本周/上周投诉量、环比增幅，超40%标「升温」标签 |
| 复盘主页 | 时段矩阵容器 | 行=公园、列=时段标签（晨/午/晚），单元格显示投诉次数，升温格橙色高亮 |
| 复盘主页 | 数据导出 | 按选中日期区间导出原始明细CSV文件 |

## 3. 核心流程

用户进入页面 → 系统自动加载本地CSV数据 → 展示默认时间范围的所有分析图表 → 用户可切换周末/施工筛选器 → 图表联动刷新 → 用户框选日期区间 → 点击导出CSV → 浏览器下载明细数据

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：藏青色 `#1e3a5f`（政务专业感），搭配橙色 `#ff6b35`（升温警告）
- **辅助色**：浅灰蓝 `#e8f0f7`，深灰 `#2d3748`
- **按钮风格**：圆角8px，悬停有轻微阴影，主按钮藏青底白字
- **字体**：使用 "Noto Sans SC" 作为中文显示字体，搭配等宽字体展示数据
- **布局风格**：卡片式布局，各模块有清晰边界，信息密度适中
- **图标风格**：使用 lucide-react 线性图标，保持简约专业

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 复盘主页 | 顶部筛选区 | 横排布局，筛选标签、日期选择器、导出按钮，背景浅灰蓝 |
| 复盘主页 | 周汇总柱状图 | 卡片容器，标题+图例在顶部，图表主体，支持X轴时间滚动 |
| 复盘主页 | 环比分析表 | 表格布局，表头固定，「升温」标签橙色圆角胶囊 |
| 复盘主页 | 时段矩阵容器 | 网格布局，行头公园名，列头时段名，单元格 hover 高亮 |
| 复盘主页 | 整体动效 | 页面加载时模块依次淡入，数据刷新时表格行淡入过渡 |

### 4.3 响应性

- 桌面端优先（1280px+）：四栏布局，矩阵完整展示
- 平板端（768px-1279px）：两栏布局，柱状图与环比表上下排列
- 移动端（<768px）：单列堆叠，矩阵横向滚动
- 所有交互元素支持触摸操作，按钮最小尺寸44px

### 4.4 视觉细节

- 柱状图柱子采用渐变填充，选中公园柱子有发光效果
- 升温单元格橙色渐变背景 + 微光动画
- 表格斑马纹提升可读性
- 卡片阴影 `0 4px 20px rgba(30, 58, 95, 0.08)`
- 数字使用等宽字体，右对齐便于对比
